# Privacy Policy for EpiQuest

Last updated: June 1, 2026

EpiQuest is a research-based educational learning game 
developed by Agamitha for academic research purposes.

## Information We Collect
- Session codes for authentication
- Learning progress and game completion data
- No personal information is collected beyond what is 
required for research purposes

## How We Use Information
- To authenticate users via session codes
- To track learning progress for research purposes
- Data is used solely for academic research by 
authorized researchers

## Data Access
- Only authorized researchers and administrators 
can access user data
- Users can only access the app with a valid session 
code provided by the administrator
